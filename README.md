# 📊 Customer Churn Prediction System

#LIVE DEMO: https://kashishcrafts-customer-churn-prediction-system-app-gql9ra.streamlit.app/

## 🚀 Overview

Customer Churn Prediction System is an AI-powered analytics dashboard that predicts whether a customer is likely to leave a company based on customer demographics, billing information, and subscribed services.

The project uses a Machine Learning Logistic Regression model trained on the Telco Customer Churn Dataset and provides real-time churn prediction through an interactive Streamlit dashboard.

---

## ✨ Features

* 🔮 Real-time Customer Churn Prediction
* 📈 Churn Probability Score
* 🚦 Risk Level Classification (Low / Medium / High)
* 💡 Automated Retention Recommendations
* 📊 Feature Importance Visualization
* 📜 Prediction History Tracking
* 📥 Download Prediction History CSV

## 📈 Important Churn Factors

### Factors Increasing Churn Risk

* Internet Service (Fiber Optic)
* Electronic Check Payment
* Paperless Billing
* Multiple Lines

### Factors Reducing Churn Risk

* Dependents
* One-Year Contract
* Two-Year Contract
* Online Security
* Tech Support
